Формула рассчета выглядит следующим образом: 
результат = (низкая частота - высокая частота)*(100/высокая частота)
Если поставить галлочку "+100 к ВП"
результат = 100+(меньшая частота - большая частота)*(100/большая частота)

Поиск происходит по координатам электродов поэтому: 
если файлы разной длины, то строчек для которых не найдено соответсвий
не будет в финальном файле



TXT
Формат входящего файла .txt:
# Rho Spa.1 Spa.2 Spa.3 Spa.4 PassTime DutyCycle Vp In Dev. K Phase Ay By My Ny

"Сохранить файл КС и ВП":
Если нажата - сохраняет файл с заголовками # Rho M Spa.1 ...
Если НЕ нажата - сохраняет файл с заголовками # Rho Spa.1 ...
Файл сохраняется с большей частотой
