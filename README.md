Всем привет! Меня зовут Степченков Владимир. Я работаю в лаборатории инженерной электроразведки в МГУ.
Для нужд 3D электротомографии я пишу эту небольшую программу. Работает программа с данными электротомографии в формате данных станции Омега-48.
Пока что программа в большей степени заточена только под многосегментную технологию Большакова Д.К., но в будущем мы это исправим для более широкой аудитории.
